# Google Drive Map

Google Drive project folder:
https://drive.google.com/drive/folders/1lvkCqFUQ1KodKogQYl8upHOGYr-7b_ra?usp=drive_link

## Purpose

Google Drive is the shared storage location for non-code assets that should be accessible across PC and Mac.

## Store In Google Drive

- Resumes
- Cover letters
- Job description backups
- Screenshots
- Exported PDFs
- Portfolio images and media assets
- Documentation files that should be easily opened outside the code editor

## Do Not Store In Google Drive

- `node_modules`
- Python virtual environments
- `.git` folders
- Build outputs
- `.env` files
- API keys
- Service account credentials
- Local dependency caches

## Suggested Drive Organization

```text
Job Search Agent OS/
  Resumes/
  Cover Letters/
  Job Descriptions/
  Screenshots/
  Portfolio Assets/
  Exported PDFs/
  Shared Documentation/
```

This map can be updated after the actual Drive folder structure is inspected or created.
