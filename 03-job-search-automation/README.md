# Job Search Automation System

## Goal

Build a human-in-the-loop job search workflow that helps find, track, analyze, and prepare job applications.

The system should make the job search more organized and faster, while keeping the user in control of every final decision.

## Intended Workflow

The system may eventually help with:
- Collecting job leads from LinkedIn and Indeed alerts.
- Tracking jobs in Google Sheets.
- Backing up job descriptions.
- Analyzing job descriptions.
- Suggesting resume tailoring ideas.
- Drafting cover letters.
- Preparing application answers.
- Tracking email updates.
- Reminding the user about next actions.

## Human Review Rule

Do not auto-submit job applications.

The system can draft, summarize, organize, and remind. The user reviews and submits manually.

## Storage Notes

- Scripts
- Prompts
- Workflow documentation
- Automation code

Google Drive:
- Resume versions
- Cover letter versions
- Job description backups
- Screenshots

- Application-related documents
